# Dynamic Efficient Frontier Lab

## Creator/Dev: Rajput Prajjval Singh

This project is a single-file Streamlit app that simulates and visualizes the efficient frontier for synthetic portfolios. It is designed to run fully offline, using only the following Python libraries: streamlit, numpy, pandas, and plotly.

## What OS is this project for?
This project is cross-platform and works on **Windows**, **macOS**, and **Linux**. You only need Python and the required libraries installed.

## How to Fork
1. Click the **Fork** button at the top right of this repository on GitHub.
2. Clone your forked repository to your local machine:
   ```sh
   git clone https://github.com/YOUR-USERNAME/YOUR-FORKED-REPO.git
   ```
3. Install the required dependencies:
   ```sh
   pip install streamlit numpy pandas plotly
   ```
4. Run the app:
   ```sh
   streamlit run app.py
   ```
